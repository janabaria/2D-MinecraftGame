# 🎮 2D Minecraft

## 📌 Overview

A simple 2D Minecraft-style game built with **HTML, CSS, and Vanilla JavaScript**.
The game lets you collect and place blocks using different tools in a small pre-built world.

---

## 🚀 How to Play

1. Select a tool
2. Click on a tile in the world
3. If the tool matches the tile → it will be collected
4. Select a block from the inventory
5. Click on the world to place it

---

## 🛠️ Tools

| Tool      | Works On     |
| --------- | ------------ |
| 🪓 Axe    | Trees        |
| ⛏ Pickaxe | Rocks        |
| 🧹 Shovel | Dirt & Grass |

> You must choose the correct tool to collect a block.

---

## 🌍 World

* The world is a 2D grid made of tiles
* Tile types:

  * Grass
  * Dirt
  * Rock
  * Tree

---

## 🎒 Inventory

* Stores collected blocks
* You can place blocks back into the world

---

## 🔄 Reset

* Resets the world to its original state
* Clears the inventory
